Follow the steps below to set up the application locally:

---

### 1. Clone the Repositories

```bash
git clone https://github.com/sahilgupta088/portfolio.git
```
*(Replace with your actual repository URL)*

---

### 2. Client Setup

```bash
cd portfolio
npm install
```

Create a `.env.local` file in the root of the client folder and add your EmailJS credentials:

```env
VITE_YOUR_SERVICE_ID=Your_EmailJS_Service_Id
VITE_YOUR_TEMPLATE_ID=Your_EmailJS_Template_Id
VITE_YOUR_PUBLIC_KEY=Your_EmailJS_Public_Id
```

Then start the client:

```bash
npm run dev
```
